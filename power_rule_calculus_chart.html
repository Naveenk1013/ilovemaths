Power rule — calculus reference chart

Everything you need to memorise from the first rule you'll use in differentiation

The core rule
d/dx [xn] = n·xn−1
f(x) = xⁿ  →  f′(x) = n·xⁿ⁻¹
Bring the exponent down as a coefficient, then subtract 1 from the exponent. Works for any real number n — positive, negative, fraction, or zero.
3-step method
01
Identify n
Spot the exponent on x. That number is n.
02
Multiply by n
Write n in front as a coefficient (×any existing coefficient).
03
Decrease n by 1
The new exponent is n − 1. That's your derivative.

Memory tips & common mistakes
Mnemonic — "Bring down, knock down": Bring the exponent down in front, knock the exponent down by 1. Two moves, done.
Common mistake — forgetting the coefficient: d/dx [4x³] ≠ 3x². You must multiply: 4 × 3 = 12, giving 12x².
Rewrite first: The rule needs the form xⁿ. Rewrite √x → x^½ and 1/x² → x⁻² before differentiating.
Term-by-term: For a sum like x³ + 5x² − 2x + 7, apply the power rule to each term separately. Constants vanish.
