Statistical Estimation

Using a small sample to predict the behavior of an entire population.

Samples vs Populations

You rarely have data for an entire population. Instead, you take a sample, calculate statistics, and use those to estimate the true population parameters.

$$ \text{CI} = \bar{x} \pm Z \left( \frac{\sigma}{\sqrt{n}} \right) $$
Confidence Interval for the Mean. Gives a range of values where the true population mean likely sits.
ConceptPoint Estimate

A single value (like sample mean) used to estimate the population value.

ConceptMargin of Error

The plus/minus amount attached to your point estimate.

Find a 95% Confidence Interval for sample mean $ \bar{x} = 50 $, standard deviation $ \sigma = 10 $, and sample size $ n = 100 $. (Use $z = 1.96$)
Step 1
Analyze the given options:
  • Option A: $ [48.04, 51.96] $
  • Option B: $ [49.02, 50.98] $
  • Option C: $ [40.00, 60.00] $
  • Option D: $ [45.00, 55.00] $
Step 2
$\text{Apply Core Principles}$
Standard error is $ SE = \frac{\sigma}{\sqrt{n}} = \frac{10}{\sqrt{100}} = 1 $. The margin of error is $ z \cdot SE = 1.96 \times 1 = 1.96 $. Thus, the 95% CI is $ [50 - 1.96, 50 + 1.96] = [48.04, 51.96] $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ [48.04, 51.96] $)

What is an unbiased estimator?
Step 1
Analyze the given options:

  • Option B: An estimator with zero variance
  • Option C: An estimator based on a huge sample size
  • Option D: A rounded value
Step 2
$\text{Apply Core Principles}$
An estimator is unbiased if its mathematical expected value is equal to the actual population parameter being estimated: $ E(\hat{\theta}) = \theta $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.

What happens to the margin of error of a confidence interval as the sample size $n$ increases?
Step 1
Analyze the given options:
  • Option A: It decreases
  • Option B: It increases
  • Option C: It remains unchanged
  • Option D: It becomes zero
Step 2
$\text{Apply Core Principles}$
Since sample size $n$ sits in the denominator of the standard error formula ($ \sigma / \sqrt{n} $), increasing $n$ shrinks the standard error, thereby decreasing the margin of error.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A (It decreases)

$\text{Apply Core Principles}$
