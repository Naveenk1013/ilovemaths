Real Number System

Learn about the continuous number line, mathematical properties, intervals, and absolute value.

What is the Real Number System?

The set of Real Numbers ($ \mathbb{R} $) includes every single number that can be plotted on an infinite, continuous number line. There are no "gaps".

$$ \text{Distance from zero: } |x| = \begin{cases} x & \text{if } x \ge 0 \\ -x & \text{if } x < 0 \end{cases} $$
The absolute value measures how far a number is from zero, ignoring its direction. Distance is always positive!
Properties Commutative & Associative

Order and grouping don't matter for addition and multiplication.

$$ a + b = b + a $$ $$ (a \times b) \times c = a \times (b \times c) $$
Analogy: Pouring milk then coffee is the same as pouring coffee then milk.
Properties Distributive Property

Multiplication distributes over addition.

$$ a(b + c) = ab + ac $$
$$ 2(x + 3) = 2x + 6 $$
Notation Open vs Closed Intervals

Parentheses () exclude the endpoints. Brackets [] include them.

$$ (a, b) \implies a < x < b $$ $$ [a, b] \implies a \le x \le b $$
Analogy: (18, 65) means "strictly between 18 and 65". [18, 65] means "from 18 to 65 inclusive".
Concept Absolute Value $|x|$

The physical distance of a number from zero on the number line.

$$ |-5| = 5 \quad \text{and} \quad |5| = 5 $$
Analogy: Walking 5 steps forward or 5 steps backward — you still walked 5 steps.

✓ Infinity is not a number
When using interval notation, you must always use an open parenthesis ) or ( next to infinity ($ \infty $) because you can never "reach" or "include" infinity. Example: $$ [0, \infty) $$.
# Absolute Value Trick
When solving an equation like $$ |x| = 7 $$, split it into two equations immediately: $$ x = 7 $$ AND $$ x = -7 $$. Both numbers are exactly 7 units away from zero.
→ Subtraction is NOT Commutative
Remember that $ a + b = b + a $, but $ a - b \neq b - a $ (unless $a=b$). E.g., $ 5 - 3 = 2 $, but $ 3 - 5 = -2 $.
