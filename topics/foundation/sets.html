Sets & Operations

Learn how to group distinct objects together, how to write them, and how they interact using unions and intersections.

What is a Set?

A set is a well-defined collection of distinct objects, usually numbers. We use curly braces {} to enclose the members (or "elements") of a set.

$$ A \cup B = \{x \mid x \in A \text{ or } x \in B\} $$
The Union of two sets contains everything in $A$ combined with everything in $B$ (without repeating elements).
Basics Roster vs Set-Builder

Two ways to define a set.

$$ A = \{1, 2, 3\} $$ (Roster)
$$ B = \{x \mid 0 < x < 4, x \in \mathbb{N}\} $$ (Set-builder)
Operation Union ($ \cup $)

Combine all elements from both sets.

$$ A \cup B $$
If $ A=\{1,2\} $ and $ B=\{2,3\} $
Then $ A \cup B = \{1,2,3\} $
Operation Intersection ($ \cap $)

Only the elements they have in common.

$$ A \cap B $$
If $ A=\{1,2,3\} $ and $ B=\{2,3,4\} $
Then $ A \cap B = \{2,3\} $
Operation Difference ($ - $)

Elements in A that are NOT in B.

$$ A - B $$
If $ A=\{1,2,3\} $ and $ B=\{2,3,4\} $
Then $ A - B = \{1\} $
Relation Subset ($ \subset $)

When all elements of one set are completely inside another.

$$ A \subset B $$
$ \{1,2\} \subset \{1,2,3\} $
Quick Revision Box
What does $\in$ mean?"Is an element of". E.g., $ 2 \in \{1,2,3\} $
What is the Empty Set?A set with zero elements, written as $ \emptyset $ or $ \{\} $.
Do sets have an order?No. $ \{1,2\} $ is the same as $ \{2,1\} $.

✓ U vs $\cap$ Memory Trick
The $\cup$ looks like a bucket holding everything (Union).
The $\cap$ looks like an archway that only lets the overlapping things through (Intersection).
# Repeats don't matter
A set only cares about distinct elements. Writing $$ \{1, 1, 2, 3, 3\} $$ is exactly the same as $$ \{1, 2, 3\} $$. Never list an element twice!
