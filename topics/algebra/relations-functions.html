Relations and Functions

Understand how inputs map to outputs, the difference between a relation and a function, and how to find domain and range.

Inputs and Outputs

A relation is just a set of ordered pairs $(x, y)$. A function is a strict type of relation where every input has exactly one output. It's like a machine: you drop a number in, and a predictable number comes out.

$$ f: X \to Y $$
"Function $f$ maps elements from set $X$ (Domain) to elements in set $Y$ (Codomain)."
Vocabulary Domain & Range

The set of all possible inputs and outputs.

Domain = All valid $x$ values
Range = All resulting $y$ values
Rule The Function Rule

One input ($x$) cannot point to two different outputs ($y$).

$$ (2, 3) \text{ and } (2, 5) \implies \text{Not a function} $$
Analogy: Pressing the "Coca-Cola" button on a vending machine shouldn't randomly give you Coke or Sprite.
Visual Test Vertical Line Test

If you draw a vertical line on a graph and it hits the curve more than once, it's NOT a function.

Quick Revision Box
Is $y = x^2$ a function?Yes. Every $x$ gives exactly one $y$.
Is $x = y^2$ a function?No. If $x=4$, $y$ can be $2$ or $-2$ (Two outputs for one input).
What's the domain of $f(x) = 1/x$?All real numbers except 0 ($x \neq 0$).

If $ f(x) = x^2 $ and $ g(x) = 2x + 1 $, find the composite function $ (f \circ g)(x) $
Step 1
Analyze the given options:
  • Option A: $ (2x + 1)^2 $

  • Option C: $ 2(x+1)^2 $
  • Option D: $ x^2(2x+1) $
Step 2
$\text{Apply Core Principles}$
The composite function is $ f(g(x)) = f(2x + 1) = (2x + 1)^2 $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ (2x + 1)^2 $)

A function $f: \mathbb{R} \to \mathbb{R}$ is defined by $ f(x) = 3x - 5 $. Find its inverse function $ f^{-1}(x) $
Step 1
Analyze the given options:
  • Option A: $ \frac{x + 5}{3} $
  • Option B: $ 3x + 5 $
  • Option C: $ \frac{x - 5}{3} $
  • Option D: $ 5x - 3 $
Step 2
$\text{Apply Core Principles}$
Set $y = 3x - 5$. Swap $x$ and $y$: $x = 3y - 5$. Solve for $y$: $ 3y = x + 5 \implies y = \frac{x + 5}{3} $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ \frac{x + 5}{3} $)

Is the relation $ R = \{(x, y) \mid x - y \text{ is divisible by 5}\} $ on integers an equivalence relation?
Step 1
Analyze the given options:
  • Option A: Yes
  • Option B: No
Step 2
$\text{Apply Core Principles}$
Yes. It is reflexive ($x - x = 0$ is divisible by 5), symmetric (if $5 | (x-y)$, then $5 | (y-x)$), and transitive (if $5 | (x-y)$ and $5 | (y-z)$, then $5 | (x-z)$).
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A (Yes)
