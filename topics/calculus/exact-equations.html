Exact Equations

When partial derivatives line up perfectly. Learn how to solve exact differential equations.

The Condition of Exactness

An equation of the form $ M(x,y)dx + N(x,y)dy = 0 $ is exact if the partial derivatives match.

$$ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} $$
If this is true, the equation is exact and can be solved by integrating.
Step 1Test for Exactness

Differentiate $M$ with respect to $y$, and $N$ with respect to $x$.

Step 2Integrate

Integrate $M$ with respect to $x$ or $N$ with respect to $y$.

$$ F(x,y) = \int M \, dx + g(y) $$

  • Option C: $ M_x + N_y = 0 $
  • Option D: $ M = N $
Step 2
$\text{Apply Core Principles}$
A differential equation is exact if and only if the partial derivative of $M$ with respect to $y$ equals the partial derivative of $N$ with respect to $x$: $ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} $.

Final Answer: Option A ($ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} $)

$\text{Apply Core Principles}$

  • Option C: $ x^2 - y^2 = C $

Step 2
$\text{Apply Core Principles}$
Integrate $M = 2x$ with respect to $x$: $ \int 2x dx = x^2 $. Integrate $N = 2y$ with respect to $y$: $ \int 2y dy = y^2 $. Combining gives the general solution: $ x^2 + y^2 = C $.

Find the general solution of the exact equation $ (y\cos x)dx + (\sin x)dy = 0 $
Step 1
Analyze the given options:
  • Option A: $ y\sin x = C $
  • Option B: $ y\cos x = C $
  • Option C: $ \sin x + y = C $
  • Option D: $ y^2 \sin x = C $
Step 2
$\text{Apply Core Principles}$
Let $M = y\cos x$ and $N = \sin x$. Note $ \frac{\partial M}{\partial y} = \cos x = \frac{\partial N}{\partial x} $. Integrate $N$ with respect to $y$: $ \int \sin x dy = y\sin x $. Taking the partial derivative with respect to $x$ matches $M$. Hence, $ y\sin x = C $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ y\sin x = C $)
