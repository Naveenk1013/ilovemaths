Applications of Integral Calculus

Calculate exact areas of weird shapes, find volumes of spun curves, and solve physics problems.

Geometry Unlocked

Integration lets us measure things that standard geometry formulas can't handle, like the area bounded by a wiggly curve or the volume of a vase.

$$ \text{Area} = \int_a^b [f(x) - g(x)] \, dx $$
Area Between Two Curves. Integrate the "top" curve minus the "bottom" curve.
ConceptVolumes of Solid of Revolution

If you spin a curve around an axis, you create a 3D solid.

$$ V = \pi \int_a^b [f(x)]^2 \, dx $$
PhysicsVelocity to Displacement

Integrate velocity with respect to time to get distance traveled.

$$ s(t) = \int v(t) \, dt $$

$\text{Apply Core Principles}$
Using the disk method, $ V = \pi \int_a^b [f(x)]^2 dx = \pi \int_0^2 x^2 dx = \pi \left[ \frac{x^3}{3} \right]_0^2 = \frac{8\pi}{3} $.

Final Answer: Option A ($ 8\pi/3 $)

What is the average value of the function $ f(x) = 2x $ over the interval $ [1, 5] $?
Step 1
Analyze the given options:
  • Option A: $ 6 $
  • Option B: $ 5 $
  • Option C: $ 12 $
  • Option D: $ 8 $
Step 2
$\text{Apply Core Principles}$
The average value formula is $ \frac{1}{b-a} \int_a^b f(x) dx $. So, $ \frac{1}{5-1} \int_1^5 2x dx = \frac{1}{4} \left[ x^2 \right]_1^5 = \frac{1}{4} (25 - 1) = \frac{24}{4} = 6 $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ 6 $)

Which integral formula represents the arc length of a curve $ y = f(x) $ from $ x = a $ to $ x = b $?
Step 1
Analyze the given options:
  • Option A: $ \int_a^b \sqrt{1 + [f'(x)]^2} \, dx $
  • Option B: $ \int_a^b [f(x)]^2 \, dx $
  • Option C: $ \int_a^b f'(x) \, dx $
  • Option D: $ \int_a^b \sqrt{1 + f(x)} \, dx $
Step 2
$\text{Apply Core Principles}$
The standard arc length formula for a differentiable function is $ L = \int_a^b \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ \int_a^b \sqrt{1 + [f'(x)]^2} \, dx $)

$\text{Apply Core Principles}$
Compute the definite integral: $ \int_0^\pi \sin x \, dx = [-\cos x]_0^\pi = -\cos \pi - (-\cos 0) = -(-1) - (-1) = 1 + 1 = 2 $.

Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
