Integration

The opposite of differentiation. Finding the area under a curve by slicing it into infinite rectangles.

Accumulation

If differentiation finds the rate of change, integration finds the accumulation of that quantity. It gives you the total area under a function's curve.

$$ \int x^n \, dx = \frac{x^{n+1}}{n+1} + C $$
The Power Rule for Integration. Add 1 to the power, divide by the new power. Always add $+C$!
ConceptIndefinite Integrals

Finding the general antiderivative formula.

$$ \int 2x \, dx = x^2 + C $$
ConceptDefinite Integrals

Finding the exact numerical area between bounds.

$$ \int_a^b f(x) \, dx $$

Analyze the given options:
  • Option A: $ 10 $

Step 2
$\text{Apply Core Principles}$
First find antiderivative: $ \int (2x + 1) dx = x^2 + x $. Now evaluate from 1 to 3: $ (3^2 + 3) - (1^2 + 1) = (9 + 3) - (1 + 1) = 12 - 2 = 10 $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ 10 $)

Evaluate by parts: $ \int x \cos x \, dx $
Step 1
Analyze the given options:
  • Option A: $ x \sin x + \cos x + C $
  • Option B: $ x \sin x - \cos x + C $
  • Option C: $ -x \sin x + \cos x + C $
  • Option D: $ x \cos x + C $
Step 2
$\text{Apply Core Principles}$
Let $u = x \implies du = dx$ and $dv = \cos x dx \implies v = \sin x$. By parts: $ \int u \, dv = uv - \int v \, du = x\sin x - \int \sin x dx = x\sin x + \cos x + C $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ x \sin x + \cos x + C $)
