Three Dimensional Geometry

Expand beyond the flat plane. Learn about coordinates, lines, and planes in 3D space.

Adding the Z-Axis

In 2D geometry, we have the x and y axes. In 3D geometry, we add depth with the z-axis, allowing us to describe points in real-world space.

$$ P(x, y, z) $$
Every point in 3D space requires three coordinates.
FormulaDistance in 3D

The 3D Pythagorean theorem.

$$ d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2} $$
GeometryEquation of a Plane

Defined by a normal vector $\langle a,b,c \rangle$ and a point.

$$ ax + by + cz = d $$

Find the direction cosines of a line that is equally inclined to the three coordinate axes.
Step 1
Analyze the given options:
  • Option A: $ \pm\frac{1}{\sqrt{3}}, \pm\frac{1}{\sqrt{3}}, \pm\frac{1}{\sqrt{3}} $
  • Option B: $ 1, 1, 1 $
  • Option C: $ 1/3, 1/3, 1/3 $
  • Option D: $ 0, 0, 0 $
Step 2
$\text{Apply Core Principles}$
Let direction cosines be $l, m, n$. Since they are equally inclined, $l^2 = m^2 = n^2$. We know $ l^2 + m^2 + n^2 = 1 \implies 3l^2 = 1 \implies l = \pm 1/\sqrt{3} $. Hence, all three are $ \pm 1/\sqrt{3} $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ \pm\frac{1}{\sqrt{3}}, \pm\frac{1}{\sqrt{3}}, \pm\frac{1}{\sqrt{3}} $)

What is the equation of a plane passing through the point $(x_1, y_1, z_1)$ with normal vector $(A, B, C)$?
Step 1
Analyze the given options:
  • Option A: $ A(x - x_1) + B(y - y_1) + C(z - z_1) = 0 $
  • Option B: $ Ax + By + Cz = 0 $
  • Option C: $ \frac{x-x_1}{A} = \frac{y-y_1}{B} = \frac{z-z_1}{C} $
  • Option D: $ x_1 x + y_1 y + z_1 z = 1 $
Step 2
$\text{Apply Core Principles}$
The standard point-normal form equation of a 3D plane is $ A(x - x_1) + B(y - y_1) + C(z - z_1) = 0 $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ A(x - x_1) + B(y - y_1) + C(z - z_1) = 0 $)
Example 3 · Concept Application
00:00
Find the distance between the parallel planes $ x + 2y - 2z + 5 = 0 $ and $ x + 2y - 2z - 7 = 0 $.
Step 1
Analyze the given options:
  • Option A: $ 4 $
  • Option B: $ 12 $
  • Option C: $ 2 $
  • Option D: $ 6 $
Step 2
$\text{Apply Core Principles}$
The distance formula between parallel planes $Ax+By+Cz+D_1=0$ and $Ax+By+Cz+D_2=0$ is $ d = \frac{|D_1 - D_2|}{\sqrt{A^2+B^2+C^2}} = \frac{|5 - (-7)|}{\sqrt{1^2+2^2+(-2)^2}} = \frac{12}{\sqrt{9}} = \frac{12}{3} = 4 $.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A ($ 4 $)

What is the shortest distance between two skew lines in space?
Step 1
Analyze the given options:
  • Option A: The length of the common perpendicular vector
  • Option B: The difference between their intercepts
  • Option C: Always zero
  • Option D: Infinite
Step 2
$\text{Apply Core Principles}$
For two non-intersecting, non-parallel lines (skew lines) in 3D, the shortest distance is the length of their common perpendicular line segment.
Step 3
Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
Final Answer: Option A (The length of the common perpendicular vector)

Cross-reference choices to select the valid deduction. Option A perfectly aligns with the principles analyzed.
