Mathematical Induction

The domino effect of mathematics. Learn how to prove that a statement is true for an infinite number of cases.

The Domino Effect

Mathematical Induction is a proof technique. Instead of proving something $1,000,000$ times, you prove two simple steps to guarantee it works forever.

$$ P(k) \implies P(k+1) $$
"If it works for a random number $k$, it MUST work for the next number $k+1$."
Step 1The Base Case

Prove the statement is true for the first number (usually $n = 1$).

Prove $ P(1) $ is true.
Analogy: Knocking over the first domino.
Step 2The Assumption

Assume the statement is true for some arbitrary number $k$.

Assume $ P(k) $ is true.
Step 3The Inductive Step

Using the assumption, prove it is true for $k+1$.

Prove $ P(k+1) $ is true.
Analogy: Showing that if any domino falls, it will knock over the next one.
